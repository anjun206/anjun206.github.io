NAT 게이트웨이: 프라이빗 서브넷에서 외부 리소스에 접근하는 통로

프라이빗 서브넷에 인터넷 데이터가 필요한데 직접 연결은 꺼려질때 사용

아웃바운드 트래픽만 가능하고 인바운드는 차단된다

```txt
✅ (프라이빗 → NAT → 인터넷)
❌ (인터넷 → NAT → 프라이빗)
```

<br>

VPC Endpoint: AWS 내부 서비스 전용 통로

외부 인터넷 연결 필요없을때, AWS 네트워크만 필요할때 쓸 수 있다

NAT 비용 안나간다
