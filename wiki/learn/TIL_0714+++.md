아무튼 오늘은 재귀함수를 배우며 하노이탑에 관해 문제를 풀어보게 되었다.<br><br>
혼자 힘으로 풀지는 못했다. 도저히 모르겠어서 말이다.<br><br>
저리 간단한 코드로 구현가능하다는 점에서 재귀함수의 힘을 온몸으로 겪었다.<br><br>


```py
# 파이썬

def move(no, x, y) -> None:    # x = 시작 기둥
    if no > 1:                 # y = 목표 기둥
        move(no-1, x, 6-x-y)   # 6 - x - y = 중간 기둥
    print(x, y)
    if no > 1:
        move(no-1, 6-x-y, y)

number = int(input())
print((2**number)-1)

if number > 20:
    exit()
else:
    move(number, 1, 3)
```
