# 가상 메모리 과제

## 보조 페이지 테이블 관리(Managing the Supplemental Page Table)

페이지 테이블의 한계 보완위해 각 페이지에 추가 데이터 제공한다

이를 보조 페이지 테이블(SPT)이라 부른다

이것도 페이지 테이블로 부르지만 헷갈리니 보조 붙여주자

SPT는 두 가지 목적에 쓰인다

1. **페이지 폴트 발생 시**

    - 폴트가 난 가상 페이지 데이터 파악

2. **프로세스 종료 시**

    - 해제할 자원 판단

<br>

### SPT의 구성

마음대로 만들면 되고 최소 두 가지 접근이 있다

- 세그먼트 기반<br>
연속 페이지 묶음 단위로 관리

- 페이지 기반<br>
개별 페이지 단위로 관리
